Reddit clone where users can post and search.

### Tech-Stack
	- Front End
		- Angular 

	- Back end
		- MongoDB with Mongoose 
		- Node.js/Express

### Requirements 
- MongoDB
- Mongoose 
- Node
- Angular.js
- Express 4.x
- NPM


### Getting Started
- Web App
  - Fork the repo
  - Clone from your fork
  - npm install
  - bower install
- Database
  - install MongoDB

- Run the App
  - node server.js
